# AI Prompt Generator

An AI-powered Prompt Generator built with **Python** and **Streamlit** that transforms simple user ideas into clear, detailed, and professional AI prompts. The application helps users create high-quality prompts for tools like ChatGPT, Gemini, Claude, and other AI models.

## Features

* ✨ Convert simple ideas into professional AI prompts
* 📝 Generate clear and well-structured prompts
* 📋 One-click copy to clipboard
* ⚡ Fast and easy-to-use interface
* 🎯 Suitable for beginners and professionals
* 💻 Built with a clean and responsive Streamlit UI

## Technologies Used

* Python
* Streamlit
* Google Gemini API (or your chosen AI model)
* python-dotenv

## How It Works

1. Enter your task or idea.
2. Click **Generate Prompt**.
3. The AI creates a detailed and optimized prompt.
4. Copy and use the generated prompt with your preferred AI model.
